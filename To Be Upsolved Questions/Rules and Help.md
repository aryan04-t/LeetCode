# Rules to Maintain "To Be Upsolved Questions" List 


- Segregate and name the .md question list files based on question rating ranges of 200 width. 
> Ex: 0 to 200.md | 1000 to 1200.md | 2400 to 2600.md | etc. 

- Inside the .md file, add a question always in this format: 
> [Question Name](Question Link) - Imp. Question Tag-1 | Imp. Question Tag-2  
> Ex: [3691. Maximum Total Subarray Value II](https://leetcode.com/problems/maximum-total-subarray-value-ii) - Sparse Tables | Segment Tree  


- `#` symbol in front of a question means that question was - `Attempted` 


- - - 


### Full Forms of Short Tags Used for Labelling Questions: 

1. DP - Dynamic Programming 
2. D&C - Divide and Conquer
